# Java Practice Repository

This repository contains Java programs written for regular practice and concept understanding.  
It includes small programs, examples, and exercises covering core Java topics.

## Topics Covered

- Java basics & syntax  
- Control statements & loops  
- Arrays & Strings  
- Object-Oriented Programming (OOP)  
- Collections Framework  
- Exception Handling  
- Multithreading basics  
- File Handling  
- Practice problems  

## Purpose

The purpose of this repository is to:
- Practice Java programming concepts  
- Improve problem-solving skills  
- Maintain a personal reference for Java topics  
- Track learning progress over time  

This repository is updated as I continue practicing and learning Java.
